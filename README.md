# ruby_toolbox

run `gem build`

then run 
`gem install ./mimo_wheel-0.0.1.gem -n /usr/local/bin`

and you can use the tool

### git
```
mimowheel [options] [commit comment]
options
   // a for add, c for commit, p for push, you can use any one of this command
   // like -a, -cp, -p and so on
   -acp, 
```
